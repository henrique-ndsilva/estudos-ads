1. Quais são os dois principais itens que são verificados na análise de viabilidade de um projeto de sistema?

Resposta Correta: Financeiros e técnicos.

2. Um diretor da área de desenvolvimento, com perfil tradicional e preocupado com os objetivos finais do software, sugeriu um modelo que combinasse os fundamentos da prototipação com os aspectos sistemáticos e controlados do modelo "cascata".

Qual o modelo sugerido pelo diretor de desenvolvimento?

Resposta Correta: Espiral

3. No algoritmo descrito abaixo, representado em pseudocódigo, que tipo de estrutura de decisão está sendo utilizada? Assinale a alternativa correta. 

início
NOME: literal
MEDIA: numérico
leia NOME
leia MEDIA
se MEDIA >= 7
então escreva "O aluno", NOME, "está aprovado"
senão escreva "O aluno", NOME, "está reprovado"
fim-se
fim

Resposta Correta: Estrutura de decisão composta.

4. O plano de gerenciamento de riscos é um dos planejamentos mais importantes que o gerente de projeto deve elaborar.

Qual das alternativas abaixo descreve os riscos mais importantes que devem ser mapeados?

Resposta Correta: Projeto, produto e negócio.

5. Um sistema da Empresa Alfa foi acessado indevidamente por usuários maliciosos e causaram prejuízos econômicos para a empresa.

Qual das alternativas abaixo descreve o principal atributo do sistema que se mostrou frágil?

Resposta Correta: Confiança e proteção.

6. O diretor de tecnologia solicitou ao gerente do projeto de desenvolvimento do software de monitoramento de remédios controlados que o algoritmo fosse representado graficamente.

Indique a alternativa que corresponde às representações gráficas de algoritmos:

Resposta Correta: Fluxograma e diagrama de Chapin.

7. O presidente de uma rede de drogarias recomendou ao diretor de tecnologia, com relação ao desenvolvimento do software de monitoramento de remédios controlados, muita atenção no gerenciamento de riscos que pudessem afetar a organização.

Qual alternativa indica o tipo de risco a que o presidente se refere?

Resposta Correta: Risco de negócios.

8. O presidente de uma rede de drogarias solicitou o desenvolvimento de um software para monitorar a venda de remédios controlados. O diretor da área de tecnologia da empresa solicitou ao gerente de projetos atenção nas tarefas de atribuição de prazos, responsáveis e recursos para cada atividade do projeto.

Qual alternativa representa a atividade de gerenciamento de projetos a que o diretor de tecnologia se refere?

Resposta Correta: Programação de atividades.

9. Durante a fase de levantamento de requisitos, o diretor de tecnologia sugeriu que fossem realizadas reuniões com os principais envolvidos no projeto, principalmente para evitar requisitos contraditórios e promover acordos entre os solicitantes.

Qual é a alternativa que indica a técnica sugerida pelo diretor?

Resposta Correta: Técnica de levantamento em grupo.

10. O gerente do projeto solicitou que no diagrama de casos de uso fosse identificada a sequência na qual os casos de uso precisam ser desenvolvidos.

Que tipo de relacionamento deve ser utilizado para atender à solicitação do gerente de projetos?

Resposta Correta: Dependência

11. Em uma reunião de controle do andamento do Projeto Pegasus, referente ao desenvolvimento de um novo software, o gerente de Projetos estava mais preocupado com as respostas para as questões do tipo?

Resposta Correta: Quanto tempo estamos atrasados? Qual o impacto no orçamento?

12. Na representação de um algoritmo, por meio do pseudocódigo, qual o comando principal caracteriza uma estrutura de decisão composta?

Assinale a alternativa correta.

Resposta Correta: "senão "

13. Entre as técnicas principais utilizadas para o levantamento de requisitos, o diretor executivo sugeriu adotar a utilização de questionários, realização de entrevistas e análise de documentação.

A qual o tipo de técnica de identificação de requisitos o diretor se referiu?

Resposta Correta: Técnicas tradicionais

14. Para o desenvolvimento do Projeto Pegasus, de desenvolvimento de software, a Diretoria de Tecnologia da Informação (TI) determinou que fosse adotada uma estratégia na qual os componentes do novo software fossem desenvolvidos como unidades independentes.

Qual paradigma está mais associado a essa estratégia?

Resposta Correta: Orientado a Objeto.

15. O produto gerado pelo Projeto Pegasus está em produção, iniciando a nova etapa do ciclo de vida do desenvolvimento: a manutenção. Porém, após a fase de testes e antes da implantação, o sistema operacional de seu servidor foi alterado sem a devida comunicação ao gestor do projeto, trazendo problemas de lentidão ao produto.

Qual o motivo principal e que tipo de manutenção deve ser aplicada?

Resposta Correta: Alterações no ambiente de processamento: manutenção adaptativa.

16. Qual alternativa descreve o modelo de desenvolvimento em que o software é produzido em uma série de versões evolucionárias e reconhece abertamente as possibilidades de algo dar errado?

Resposta Correta: Espiral.

17. A equipe do projeto do desenvolvimento do software solicitou que fosse desenhado o diagrama de classes.

Qual destas alternativa representa o principal objetivo do diagrama de classes?

Resposta Correta: Fornecer uma representação gráfica da estrutura estática do software.
