7. Elabore um programa em linguagem de programação C# que informa o maior número inteiro dentre 10 números inteiros informados pelo usuário:

using System;

int num, numMaior;

Console.WriteLine("Digite um número: ");
numMaior = int.Parse (Console.ReadLine());

for (int i = 1; i < 10; i++) {
    Console.WriteLine("Digite um número: ");
    num = int.Parse (Console.ReadLine());

    if (num > numMaior)
        numMaior = num;
}

Console.WriteLine("Numero maior é: " + numMaior);

Explicação: O algoritmo lê 10 números inteiros digitados pelo usuário e determina qual deles é o maior. O primeiro número é armazenado como o maior inicialmente. Em seguida, os próximos 9 números são comparados com esse valor, e, se algum for maior, ele substitui o anterior. Ao final, o maior número informado é exibido na tela.
